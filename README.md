# Bhojan Box - Food Delivery Website
A fully responsive Swiggy Clone built using React.js with modern frontend development practices and state management using Redux.
This project replicates the core functionality of a food delivery platform including restaurant listings, menu browsing, cart management, routing, and optimized performance techniques like lazy loading.

# Live Demo

https://bhojan-box.vercel.app/

# Functionalities
🍕 Browse restaurants and food items
🛒 Add / Remove items from cart
🔄 Global state management using Redux
⚡ Lazy Loading for performance optimization
🧭 Dynamic Routing with React Router
🎣 React Hooks implementation
🧱 Functional Components
🏛️ Class-Based Components
📱 Responsive UI Design
🔍 Clean and modular project structure
